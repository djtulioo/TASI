# **Project Strategy and Action Model Canvas**

Referência: Canvas de Objetivos de Projeto (Imersão)

Projeto: Sistema Pulsar

| Seção | Detalhes |
| :---- | :---- |
| **Visão Geral do Desafio** | As instituições sofrem com feedback fragmentado (e-mail, corredor, redes sociais), gerando dados desestruturados que impedem uma visão clara dos problemas reais da organização. |
| **Objetivo Principal (Smart Goal)** | Desenvolver e implantar, até o final do semestre, uma plataforma omnicanal que centralize 100% dos feedbacks digitais e utilize IA para reduzir em 80% o tempo de triagem das reclamações. |
| **Objetivos Específicos** | 1\. **Redução de Atrito:** Implementar coleta via Telegram/WhatsApp para aumentar o engajamento. 2\. **Análise Automatizada:** Usar LLMs para classificar sentimento e tópicos automaticamente. 3\. **Acessibilidade:** Substituir dashboards complexos por uma interface de chat em linguagem natural. |
| **Indicadores de Sucesso (KPIs)** | 1\. **Tempo de Resposta:** Reduzir de 48h para \<1h a notificação de problemas críticos. 2\. **Acurácia do RAG:** \<5% de alucinações nas respostas da IA. 3\. **Volume:** Aumentar em 30% a coleta de feedbacks comparado aos métodos físicos. |
| **Premissas e Restrições** | Baixo custo (Google Cloud Run \+ Gemini Flash); Anonimização obrigatória de dados sensíveis (LGPD). |
